# PulsePoint Community System

## Description
A web-based system for managing and viewing community announcements, events, polls, and feedback. Supports three actors: Residents (mobile and display) and Admins.

## Features
- Resident: View announcements/events, participate in polls, submit feedback
- Admin: Manage announcements, events, polls, and feedback
- Public display mode for community spaces
- Mobile QR-based resident access

## Installation
1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm start` to launch the app

## Usage
- Access `/display` for public display
- Access `/mobile` via QR code
- Admin panel at `/admin` (requires login)
